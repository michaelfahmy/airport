# Airport Management Portal

## Requirements:
* Ruby 2.6.3
* Rails 5.2.3
* Postgresql
* Node ~> 11.10.0
* Yarn ~> 1.13.0

## Setup
* `bundle`
* Configure db with your creds in `config/database.yml`
* `rails db:create db:migrate db:seed`
* You're good to go now ;)

## Development
Run `rails s`

## Testing
Run `rspec`

## Docs
Visit `/docs`
